This exercise covers:
- URI template patterns and parameter substitution
- Dynamic resource generation
- Template matching and validation
- Resource template management

### Instructions

1. Open `exercise_09.py`
2. Complete all the TODO comments

### What you'll practice

- **URI templates**: Parameterized resource URIs
- **Template expansion**: Converting templates to actual URIs
- **Dynamic content**: Generating resources on demand
- **Template validation**: Ensuring template correctness

### Key Concepts

- **URI templates**: Patterns like `config://app/{section}/{key}`
- **Parameter substitution**: Filling template placeholders
- **Template matching**: Identifying which template matches a URI
- **Dynamic generation**: Creating content based on URI parameters
