## Exercise 24: Complete MCP Server

This exercise covers all MCP features in a comprehensive project manager implementation.

### Key Concepts Covered

- **Resources**: Project and task data access
- **Tools**: CRUD operations with authentication
- **Prompts**: Dynamic prompt generation
- **Elicitation**: User confirmations for destructive operations
- **Authentication**: Token-based security
- **Data Models**: Structured project and task management

### Instructions

1. Open `exercise_24.py`
2. Study the complete `ProjectManager` class implementation
3. Run the test: `python3 test_exercise_24.py`
4. Understand how all MCP features work together

### Features Demonstrated

- Project and task CRUD operations
- Resource listing and reading
- Authentication and authorization
- Prompt generation for different scenarios
- Elicitation for confirmations
- Error handling and validation
