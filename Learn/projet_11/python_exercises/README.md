This exercise covers:
- Creating prompt management systems
- Defining prompt metadata (name, title, description)
- Managing prompt messages
- Organizing multiple prompts

### Instructions

### Key Concepts

- **Prompts**: Predefined message templates for LLMs
- **Messages**: List of user/assistant messages
- **Metadata**: Information about each prompt (name, description, etc.)
- **Management**: Organizing and retrieving prompts
