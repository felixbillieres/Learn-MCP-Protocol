# Project 35 : Malware Analyzer

## Objective

Create an MCP server to analyze suspicious files and generate YARA rules.

## Concepts to Learn

### Malware Analysis

Malware analysis involves:
- **Static Analysis** : Analyze file without execution
- **Dynamic Analysis** : Execute in sandbox
- **YARA Rules** : Pattern matching for malware detection
- **Threat Intelligence** : Compare with known malware

### MCP Architecture

- **Tools** : Analyze files
- **Resources** : Expose analysis reports
- **Prompts** : Generate YARA rules
- **Elicitation** : Confirm before analysis

## Use Cases

- Analyze suspicious files
- Generate detection rules
- Compare with threat intelligence
- Document malware characteristics
